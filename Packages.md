# Packages

## config

- [config](https://www.npmjs.com/package/config)

在 config 目录中写好配置文件，例如 `default.json` `production.json` `development.json` ...  启动时根据 NODE_ENV 的值来根据配置文件，default.json 作为默认加载的配置，其他的配置文件是拓展。
